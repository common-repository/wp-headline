== Description ==

Adds headline BBCodes to WordPress.

With index:
[hl]Headline[/hl]
[hl=1-5]Headline[/hl]

Without index:
[h]Headline[/h]
[h=1-5]Headline[/h]

== Installation ==

Upload the plugin to your blog, Activate it.
